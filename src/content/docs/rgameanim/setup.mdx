## Project Settings

RGameAnim requires the following data is added to project config `.ini` files found in the `(ProjectName)/Config/` folder.

### `DefaultEngine.ini`

<Steps>

1. Add the following `ConsoleVariables` to the projects `(ProjectName)/Config/DefaultEngine.ini` file.

</Steps>

<Steps>

3. Update Project Settings `Project/Maps & Modes/Default GameMode` to `GM_Sandbox`. You can also just update the `World Settings` tab `GameMode Override` for a specific level/map.

</Steps>
